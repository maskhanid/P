Kebijakan Privasi & Ketentuan Penggunaan

Developer: Mas Khan ID

1. Pengenalan

Aplikasi ini dirancang untuk membantu pengguna melakukan pemantauan perangkat jarak jauh secara efektif. Namun, kami tidak bertanggung jawab atas segala bentuk penyalahgunaan aplikasi ini. Pengguna harus menggunakan aplikasi ini dengan bijak dan sesuai hukum yang berlaku.

2. Kebijakan Privasi

Kami menghormati privasi pengguna. Aplikasi ini mengumpulkan data hanya untuk keperluan operasional aplikasi. Data tersebut meliputi:

  • Data perangkat target (seperti log aktivitas dan lokasi)
  • Informasi login pengguna

Data yang dikumpulkan akan dilindungi dengan teknologi keamanan terbaru dan tidak akan dibagikan kepada pihak ketiga tanpa persetujuan pengguna, kecuali diwajibkan oleh hukum.

3. Ketentuan Penggunaan

Dengan menggunakan aplikasi ini, Anda setuju untuk mematuhi ketentuan berikut:

  • Aplikasi hanya boleh digunakan untuk tujuan yang sah.
  • Dilarang menggunakan aplikasi untuk melanggar privasi orang lain tanpa izin.
  • Kami tidak bertanggung jawab atas kerugian yang disebabkan oleh penyalahgunaan aplikasi.

4. Disclaimer

Aplikasi ini bersifat "as is" tanpa jaminan apa pun. Pengguna bertanggung jawab sepenuhnya atas konsekuensi dari penggunaan aplikasi. Kami tidak menganjurkan penggunaan aplikasi ini untuk aktivitas ilegal.

5. Perubahan Kebijakan

Kami berhak mengubah kebijakan privasi dan ketentuan penggunaan sewaktu-waktu. Pengguna disarankan untuk memeriksa halaman ini secara berkala.
